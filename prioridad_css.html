PRIORIDAD DE ESTILOS

!important

Dentro del código CSS, en caso de conflicto entre dos reglas o declaraciones, la última escrita es la que tiene prioridad.

HTML
<p>Un párrafo</p> <p>Otro párrafo<p>
CSS
p { font-family: arial; color: blue; } p { font-family: monospace; color: red; }

Un párrafo

Otro párrafo

Sin embargo podemos alterar este comportamiento utilizando el indicador !important junto con el valor de una declaración para establecerla como prioritaria aunque otra regla posterior debiera anularla.

La sintaxis sería la siguiente: propiedad: valor !important;

Veamos como cambia el ejemplo anterior utilizando !important

HTML
<p>Un párrafo</p> <p>Otro párrafo<p>
CSS
p { font-family: arial; color: blue !important; } p { font-family: monospace; color: red; }

Un párrafo

Otro párrafo

Aunque la tipografía de la primera regla se ha "machacado" con la de la segunda, se mantiene el color de la primera.


Orden de prioridad de estilos CSS

La siguiente pirámide indica que estilo CSS se aplica por encima de otros:

CSS del usuario (navegador) + !important
CSS declarado o vinculado en HTML + !important
CSS declarado o vinculado en HTML
CSS del usuario (navegador)
CSS por defecto del navegador

Prioridad según la especifidad de las reglas CSS

Se puede dar el caso de que varias reglas de estilo se apliquen sobre un mismo elemento entrando en conflicto sus declaraciones.

Para saber que regla se aplicará hay que compararlas según lo especificas que sean según los siguientes parámetros:

  1. Si se trata de un estilo en línea - La que más tenga, normalmente sólo una, es la que se aplica. En caso de empate se pasa al siguiente parámetro.
  2. Cuantos id's contiene el selector de cada regla - la que más tenga se aplica. En caso de empate...
  3. Cuantas clases contiene el selector de cada regla - la que más tenga se aplica. En caso de empate...
  4. Cuantos elementos HTML contiene el selector de cada regla - la que más tenga se aplica. En caso de empate se aplica la última regla declrada (ver el primer ejemplo de la página).

Un ejemplo para entenderlo:

HTML
<body> <div id="contenedor"> <header class="cabecera"> <h1>Encabezado</h1> </header> </div> </body>
CSS
/*Regla1*/ #contenedor header h1 { color: green; } /*Regla2*/ #contenedor .cabecera h1 { color: red; } /*Regla3*/ header h1 { color: blue; }
COMPARACIÓN
Parámetro 1: R1=0 R2=0 R3=0
Parámetro 2: R1=1 R2=1 R3=0, R3 descartada
Parámetro 3: R1=0 R2=1, R1 descartada
