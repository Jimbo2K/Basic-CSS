Que es un pseudoelemento

Los pseudoelementos se utilizan junto con los selectores para modificar el estilo de una parte concreta del elemento HTML al que se le está aplicando la regla.

La sintáxis consiste en añadir al final del selector "::nombre_pseudoelemento".

Ejemplo
#cajaejemplo > div p::before { content: "Ejemplo -" font-weight: bold; }

::before

Este pseudoelemento se utiliza junto con la propiedad content para añadir al elemento (indicado en el selector) un contenido antes del que ya se ha declarado en el HTML

El contenido puede ser texto, el valor de un atributo del elemento, una URL que contenga una imagen, un contador ... Para más información: content en w3schools.

Junto con content se pueden añadir otras declaraciones para aplicar estilos al contenido a añadir.

HTML
<div> <p>Resultado</p> <p id="psbefore">Un párrafo para comenzar</p> </div>
CSS
#psbefore::before { content: "Inicio - "; color: green; font-weight: bold; }

Resultado

Un párrafo para comenzar

::after

Funciona de manera idéntica a ::before salvo que añade el valor de content después del contenido del elemento HTML que indica el selector.

::first-letter

Permite modificar el estilo de la primera letra del elemento HTML al que apunta el selector.

Evidentemente se utiliza con elementos que contengan texto.

HTML
<div> <p>Resultado</p> <p id="psfletter">En un lugar de la Mancha</p> </div>
CSS
#psfletter::first-letter { font-family: "Times New Roman"; font-size: 2em; padding-right: 2px; text-transform: uppercase; }

Resultado

En un lugar de la Mancha

::first-line

Funciona de manera similar a ::first-letter pero en lugar de afectar a la primera letra actua sobre la primera línea de texto, sea la que sea.

El último comentario se refiere a que al modificar el tamaño de centana del navegador las líneas de texto de las cajas se suelen modificar.

::selection

Modifica el estilo que adquiere el texto de un elemento HTML cuando es seleccionado en pantalla con el cursor.

Este pseudoelemento no es compatible con Firefox por lo que tendremos que añadir una regla utilizando su vendro prefix: selector::-moz-selection.
