Que es una pseudoclase

Las pseudoclases permiten modificar el estilo de un elemento en función de su estado o posición jerárquica (último hijo, 5º descendiente, etc.).

La sintáxis consiste en añadir al final del selector ":nombre_pseudoclase".

Ejemplo
#cajaejemplo > div p:first-child { color: red; font-style: italic; }

Pseudoclases Links

Se utilizan principalmente con elementos <a>.

:link

Permite modificar el estilo de enlaces <a> que todavía no han sido visitados por los usuarios de la Web.

HTML
<div> <a href="http://www.google.es">Google</a> <a href="http://www.w3schools.com" class="pslink">w3schools</a> </div>
CSS
a.pslink:link { color: red; }
RESULTADO
Haz click en los siguientes enlaces:

Google

w3schools

Nota: si quieres "desvisitar" los enlaces tendrás que borrar los datos de navegación.

:visited

Modifica el estilo de enlaces que han sido visitados por los usuarios de la Web.

HTML
<div> <a href="http://www.google.es">Google</a> <a href="http://www.wikipedia.org" class="psvisited">Wikipedia</a> </div>
CSS
a.psvisited:visited { color: green; }
RESULTADO
Haz click en los siguientes enlaces:

Google

Wikipedia

Nota: si quieres "desvisitar" los enlaces tendrás que borrar los datos de navegación.

:hover

Modifica el estilo de un elemento mientras el cursor esté sobre él.

HTML
<div> <p>Un párrafo normal.</p> <p class="pshover">Un párrafo especial.</p> <p>Otro párrafo normal.</p> </div>
CSS
p.pshover:hover { color: yellow; }
RESULTADO
Pasa el cursor sobre los párrafos

Un párrafo normal.

Un párrafo especial.

Otro párrafo normal.

:active

Modifica el estilo de un elemento mientras se esté haciendo click sobre él. Su uso tiene un sentido claro para <a> pero también se puede utilizar con otros elementos html.

HTML
<div> <p id="secreto">¡¡SORPRESA!!</p> </div>
CSS
#secreto { color: red; background-color: red; } p#secreto:active { background-color: #ccc; }
Resultado
Haz click sobre el recuadro rojo

¡¡SORPRESA!!

Uso de las pseudoclases Link con enlaces <a>

Si utilizamos las pseudoclases Link para definir el comportamiento-estilo de los enlaces, debemos declarar las reglas en el orden adecuado:

  1. :link
  2. :visited
  3. :hover
  4. :active

CSS
div#enlaces a:link { text-decoration: none; color: #333; } div#enlaces a:visited { color: #777; } div#enlaces a:hover { color: lightyellow; } div#enlaces a:active { color: yellow; font-weight: bold; }
RESULTADO

Enlace 1 Google

Enlace 2 w3schools

Enlace 3 Wikipedia

Enlace 4 Aquí

Nota: si quieres "desvisitar" los enlaces tendrás que borrar los datos de navegación.

:target

Permite asignar un estilo a un elemento que sea destino de un enlace <a> dentro del mismo documento. El estilo se aplicará en el momento que pinchemos en el enlace.

Su sintáxis es un poco distinta a la normal ya que no va concatenada a un selector: div > :target {}, .clase section > ul :target{} ...

HTML
<div id="pstarget"> <p id="prf1">Este párrafo contiene el ancla 1</p> <p id="prf2">Este párrafo contiene el ancla 2</p> <br><br> <a href="#prf1">Ir al ancla 1</a><br> <a href="#prf2">Ir al ancla 2</a> </div>
CSS
#pstarget :target { border: 1px solid #d4d4d4; background-color: lightyellow; }
RESULTADO

Este párrafo contiene el ancla 1

Este párrafo contiene el ancla 2



Ir al ancla 1
Ir al ancla 2

Pseudoclases "variadas"

:lang

Esta pseudoclase permite alterar el estilo de elementos HTML en los que se haya declarado el atributo lang.

HTML
<div id="pslang"> <p lang="es_ES">Esto es un bonito párrafo en español</p> <p lang="en">This is a pretty paragraph in english</p> <p>Esto es un fárrafo en farfullo</p> </div>
CSS
div#pslang > p:lang(es_ES) { color: red; } div#pslang > p:lang(en) { color: blue; }
RESULTADO

Esto es un bonito párrafo en español

This is a pretty paragraph in english

Esto es un fárrafo en farfullo

:empty

Esta pseudoclase nos permite aplicar reglas de estilo a elementos HTML que no tengan hijos ni contenido definido.

HTML
<div id="psempty"> <div></div> <div>Hola</div> <div><p>Mundo</p></div> </div>
CSS
#psempty div { width: 60px; height: 60px; background-color: lightblue; border: 1px solid blue; } #psempty div:empty { background-color: blue; border: 2px solid lightblue; }
RESULTADO
Hola

Mundo

:not(selector)

Aplica un estilo a cualquier elemento que no sea del tipo indicado por selector.

Su sintáxis varía del resto de pseudoclases porque en lugar de ser selector:pseudoclase es :pseudoclase(selector).

HTML
<div id="psnot"> <span>Un bonito span</span><br> <p>Un párrafo normal</p> <div>Un simple div</div> </div>
CSS
#psnot :not(p) { background-color: yellow; }
RESULTADO

Un bonito span

Un párrafo normal

Un simple div

:root

Esta pseudoclase aplica un estilo al elemento raiz del documento que por tanto heredarán sus descendientes. En HTML el elemento es siempre <input>.

Esta característica hace que para su sintáxis no sea necesario escribir ningún selector: :root {propiedad: valor; ...}

Pseudoclases para formularios

Son pseudoclases que se aplican normalmnete a elementos de formularios aunque algunas se pueden utilizar en otros elementos.

:focus

Altera el estilo de un campo de entrada <input> de un formulario cuando tiene el foco de atención, es decir, cuando se ha pinchado sobre él o accedido tabulando.

:checked

Permite alterar el estilo de elementos <input> del tipo radio buttons y checkboxes, y de elementos <option>.

:disabled

Se utiliza en reglas CSS para afectar a elementos con el atributo disabled activado. Normalmente este atributo se utiliza en elementos HTML de formulario.

:enabled

Con este selector podemos modificar el estilo de elementos que utilicen el atributo disabled pero no lo tengan activado.

:in-range

Permite aplicar estilo a elementos cuyo contenido sea un valor DENTRO de un rango declarado mediante los atributos min y max.

:out-of-range

Permite aplicar estilo a elementos cuyo contenido sea un valor FUERA de un rango declarado mediante los atributos min y max.

:invalid

Aplica un estilo a elementos cuyo contenido no sea valido para el tipo de elemento: valores fuera de un rango específico, texto en un campo para números, dirección e-mail no valida ...

Por la definición anterior :out-of-range es una "variante" más específica de :invalid.

:valid

Esta pseudoclase aplica un estilo a un <input> cuyo valor/contenido sea válido.

:optional

Altera el estilo de un campo de entrada <input> que no es obligatorio rellenar, es decir, que no utiliza el atributo required.

Todos los <input> son por defecto opcionales.

:required

Permite aplicar estilo a elementos campo de entrada que tienen declarado el atributo required.

:read-only

Aplica estilo a elementos campo de entrada que tienen declarado el atributo readonly.

Pseudoclases jerárquicas

:first-child

Permite aplicar una regla a un elemento que sea el primer hijo de su elemento padre.

HTML
<div id="psfchild"> <p>El primer p hijo</p> <p>El segundo p hijo</p> </div>
CSS
#psfchild p:first-child { color: red; font-weight: bold; }
RESULTADO

El primer p hijo

El segundo p hijo

Ser el primer hijo implica que:

  • No puede tener ningún hermano que le preceda, aunque sea otro tipo de elemento HTML.
  • Tampoco puede haber ningún tipo de contenido antes.

CSS
div p:first-child { color: red; }
HTML
<div> <div></div> <p>¿Se aplicará estilo a este párrafo?</p> </div> No se aplicará la regla a <p>
HTML
<div> Esto es un div <p>¿Se aplicará estilo a este párrafo?</p> </div> No se aplicará la regla a <p>

:first-of-type

Permite aplicar estilo a un elemento que sea el primer hijo de su tipo de un elemento padre.

Se diferencia de :first-child porque el elemento determinado en el selector puede tener hermanos anteriores a él siempre que no sean de su mismo tipo.

HTML
<div id="psftype"> Esto es un div<br> <div></div> <p>¿Se aplicará estilo a este párrafo?</p> </div>
CSS
#psftype p:first-of-type { color: red; }
RESULTADO

Esto es un div

¿Se aplicará estilo a este párrafo?

:last-child

El caso contrario de :first-child, aplica estilo a un elemento (indicado en el selector) que además es el último hijo de un elemento padre.

Después del elemento no puede haber ningún otro hermano ni contenido.

:last-of-type

Aplica un estilo a un elemento que sea el último hijo de su tipo de un elemento padre, el caso opuesto a :first-of-type.

:nth-child(n)

Permite aplicar un estilo a un elemento que sea el n-esimo hijo de un elemento padre.

El valor n puede ser:

  • Un valor entero (1, 2, 3 ...)
  • El término even para indicar un grupo de elementos hijos pares
  • El término odd para indicar un grupo de elementos hijos impares
  • Una fórmula (An+B) donde podemos dar valores a A y B y n es un contador (0,1,2...)

Si utilizamos las últimas tres opciones podemos aplicar el mismo estilo a varios elementos del mismo tipo indicado en el selector.

Se comporta como :first-child y :last-child así que el contenido y otros elementos distintos del tipo del de el selector se tienen en cuenta a la hora de contar posiciones.

HTML
<div id="psnchild">RESULTADO<br> <p>Párrafo 1</p> <p>Párrafo 2</p> <p>Párrafo 3</p> <p>Párrafo 4</p> <p>Párrafo 5</p> <p>Párrafo 6</p> <p>Párrafo 7</p> </div>
CSS
#psnchild p:nth-child(even) { color: green; } #psnchild p:nth-child(2n+1) { background-color: lightblue; }
RESULTADO

Párrafo 1

Párrafo 2

Párrafo 3

Párrafo 4

Párrafo 5

Párrafo 6

Párrafo 7

Hay que tener en cuenta que en el código HTML RESULTADO que es contenido cuenta como un hijo de <div id="psnchild">, por eso los párrafos númerados como pares para CSS son realmente impares.


:nth-last-child(n)

Se comporta de manera similar a :nth-child() salvo que empieza a contar desde el último hijo hasta el primero.

:nth-of-type(n)

Aplica estilos a un elemento que sea n-esimo hijo pero a diferencia de :nth-child sólo tiene en cuenta los elementos que sean del mismo tipo.

Como en :nth-child podemos utilizar distintos valores para n: enteros, even, odd o una fórmula.

HTML
<div id="psntype">RESULTADO<br> <p>Párrafo 1</p> <span>Span 1</p><br> <p>Párrafo 2</p> <div>Div 1</div> <span>Span 2</p><br> <p>Párrafo 3</p> <p>Párrafo 4</p> </div>
CSS
#psntype p:nth-of-type(even) { color: red; }
RESULTADO

Párrafo 1

Span 1

Párrafo 2

Div 1
Span 2

Párrafo 3

Párrafo 4

:nth-last-of-type

Como :nth-of-type pero empezando a contar desde el último hijo hasta el primero.

:only-child

Con esta pseudoclase podemos aplicar estilos a elementos que son hijos únicos de un padre, que no tienen ningún tipo de hermano, ni siquiera de un tipo distinto incluido contenido del padre.

:only-of-type

Permite aplicar estilo a elementos que son hijos únicos de su tipo, es decir, que no tienen hermanos de su mismo tipo definido en el selector pero si pueden tener hermanos de otros tipos.
